[{c1}]

Present Perfect

[{p1}]

Structure: Sub + have/has + V3 + Object

Example :
  1. I have read the book. --> [{p2}]
  2. He/She has done it. --> [{p3}]
  3. They have made the garden. --> [{p4}]
  4. The work is completed.He has completed it. --> [{p5}]

[{pc1}]

Structure: Sub + have/has + been + v1-ing + object

  2. It has been raining since morning. --> [{pc3}]
